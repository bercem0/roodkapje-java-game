## 🎮 Project Overzicht

Roodkapje is een op Greenfoot gebaseerde Java-game die is ontwikkeld om objectgeoriënteerde programmeerprincipes toe te passen in een gestructureerde en interactieve omgeving. Het project demonstreert de praktische implementatie van een klasse-gebaseerde architectuur, event-driven programmering en de ontwikkeling van game-mechanieken.

## 🛠 Technische Implementatie

De game is ontwikkeld in Java binnen het Greenfoot-framework en bestaat uit meerdere aangepaste klassen die game-objecten vertegenwoordigen, zoals het spelerspersonage, interactieve items, obstakels en omgevingsobjecten.

Belangrijke technische aspecten:

- Objectgeoriënteerd programmeren (OOP) met meerdere samenwerkende klassen  
- Overerving en encapsulatie voor gestructureerd objectbeheer  
- Event handling voor gebruikersinput (toetsenbordbesturing)  
- Collision detection tussen game-objecten  
- Game state management voor levelprogressie  
- Eigen logica voor itemverzameling en levelvoltooiing  
- Afbeeldingsgebaseerde weergave van interactieve elementen  

## ⚙ Architectuur

Het project volgt een klasse-gebaseerde architectuur waarbij elk game-component als een onafhankelijk object wordt geïmplementeerd. De World-klasse beheert de spelomgeving, terwijl Actor-subklassen verantwoordelijk zijn voor spelerinteractie en objectgedrag.

Assets zoals afbeeldingen zijn geïntegreerd in het project en gekoppeld aan de bijbehorende objecten om een visuele spelervaring te creëren.

## 🎯 Doel van het Project

Dit project versterkt het begrip van:

- Java-programmeerconcepten  
- Fundamentele principes van game development  
- Interactieve systeemontwikkeling  
- Logisch probleemoplossend denken  
- Gestructureerde softwareontwikkeling
